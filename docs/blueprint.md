# **App Name**: Hafiz Ali Zaib Portfolio

## Core Features:

- Hero Section Parallax Animation: Create a visually engaging hero section with layered parallax animations on scroll, enhancing the immersive experience.
- Glassmorphism UI: Implement a sleek glass morphism aesthetic for all cards and buttons, providing a modern, frosted-glass look that enhances visual appeal and usability.
- Project Category Filter: Enable dynamic filtering of project cards based on category (Logos, Posters, UI Design, Brands Managed) to allow users to quickly find relevant work.
- Responsive Masonry Grid: Display projects in a responsive masonry grid that adapts to different screen sizes, ensuring optimal viewing on all devices.
- Featured Case Study Showcase: Highlight a specific project (Logo Design) with a dedicated section featuring a slider or side-scroll gallery to present detailed visuals and descriptions.
- Contact Form with Glassmorphism: Provide a clean, minimal contact form with glassmorphism input fields to encourage user interaction and inquiries.

## Style Guidelines:

- Primary color: Deep charcoal gray (#222222) for a premium and sophisticated background.
- Accent color: Electric cyan-blue (#00D4FF) for interactive elements and highlights.
- Card color: Semi-transparent white/light gray (rgba(255, 255, 255, 0.1) with backdrop-filter: blur(10px)) for glassmorphism effect.
- Body font: 'Inter' (sans-serif) for clean and modern body text.
- Headline font: 'Poppins' (sans-serif) for bold and elegant headings.
- Implement a responsive 12-column grid for consistent layout structure across all sections.
- Use CSS transform: translateY() for smooth, layered movement of parallax elements.
